lamp project

Titled "Foodie", this was my final project for my Database Implementation class. The project was to create a real life application of user-interface and database interaction. For my project, I decided to make a mock application for restaurant and food search. The user could use the search tool by various categories, such as price range, cuisine, and zip code. In addition, through a password protected section of the application, a user could enter a restaurant or a food dish into the database.
The application was powered mainly by PHP, MySQL, and Python.

elizabeth-ha.com

Creating a personal website has been a long term journey for me. I struggled between creating something that represented my personality but also a professional looking website for career related situations. There were a few trial and errors - the first personal website I created was quite... fruity. I made that website soon after I joined Avex Designs. I was excited to put the new skills I had learned to use.
However, as time passed, I got a better understanding of site flow, design, and typography. I created version 2, this website, after deciding on a more sophisticated concept and refining my design. I also upgraded the Javascript and jQuery that powers the interactions throughout the site. With my current site, I was able to find a way to showcase my style and also my skills as a web developer.
